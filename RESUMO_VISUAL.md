## 🎮 Como Funciona o Quiz

```
ANTES:
┌──────────────────────────────────┐
│ Questão 1: Quantas patas?        │
├──────────────────────────────────┤
│ ○ 4    ○ 6    ○ 8               │
├──────────────────────────────────┤
│ Questão 2: Qual inseto...        │
├──────────────────────────────────┤
│ ○ Abelha   ○ Piolho   ○ Pulga   │
└──────────────────────────────────┘
      ↓ Usuário seleciona respostas ↓

DEPOIS (Ao clicar em "Ver Resultados"):
┌──────────────────────────────────────┐
│         ✓ RESULTADO                  │
├──────────────────────────────────────┤
│ ✓ Questão 1 [CERTO]                 │
│   Sua resposta: 6                    │
│                                      │
│ ✗ Questão 2 [ERRADO]                │
│   Sua resposta: Piolho               │
│   Resposta correta: Abelha           │
│                                      │
│ TOTAL: 3 de 4 respostas corretas    │
└──────────────────────────────────────┘
```

## 🚀 Fluxo de Uso

```

   ↓

   ↓

   ↓

   ↓

   ↓

   ↓
```
